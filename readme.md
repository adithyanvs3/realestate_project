## Real Estate Landing Page

[live@](https://realestate-project-self.vercel.app/)

A modern and visually appealing Real Estate Landing Page built with HTML and Tailwind CSS, designed to highlight properties, feature sections, testimonials, and subscription modules. The layout is fully responsive and crafted to feel clean, elegant, and professional.

##  Features

- Beautiful hero section with blended background image

- Smooth and responsive navigation bar

- Modern property cards with pricing & detail buttons

- Multiple feature sections showcasing property highlights

- Newsletter subscription UI

- Fully responsive design for all screen sizes

- Built using Tailwind CSS for fast and customizable styling

# Technologies Used

- HTML5

- Tailwind CSS

- Font Awesome Icons
